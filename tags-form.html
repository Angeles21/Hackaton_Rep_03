Elementos de un Formulario

Al momento de crear un formulario en HTML5, es importante recordar que el usuario necesita ser guiado de una manera orgánica e inuitiva. Mientras menos esfuerzo efectue, es más probable que se encuentre dispuesto a rellenar con mayor tino ciertos campos que impliquen colocar información escrita (como opinones, comentarios, direcciones,etc).

A continuación, una serie de ejemplos con los elementos más básicos y sus respectivos ejemplos.

Form

Representa la sección del proyecto donde se colocarán todos los campos que el usuario tendrá que rellenar. Contiene una gran cantidad de elementos. Se presenta dentro del código con la palabra FORM dentro de llaves.

Label & Input

Ambos son eelementos HTML que usan para crear campos, a fin de recolectar información por parte del usuario. Dentro de ellos, contiene una gran variedad de tipos de entradas. Label tiene como funcion primaria el representar como una etiqueta (por ello su nombre), e Input es el que posee controles para recibir cierto tipo de información específica.
