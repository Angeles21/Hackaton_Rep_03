Elementos de un Formulario

Al momento de crear un formulario en HTML5, es importante recordar que el usuario necesita ser guiado de una manera orgánica e inuitiva. Mientras menos esfuerzo efectue, es más probable que se encuentre dispuesto a rellenar con mayor tino ciertos campos que impliquen colocar información escrita (como opinones, comentarios, direcciones,etc).

A continuación, una serie de ejemplos con los elementos más básicos y sus respectivos ejemplos.

Input

Es el elemento HTML que usa para crear campos, a fin de recolectar información por parte del usuario. Dentro de el, contiene una gran variedad de tipos de entradas.
